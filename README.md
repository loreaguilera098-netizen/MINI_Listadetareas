# ✅ Mi Lista de Tareas con Metas

Una mini-aplicación web interactiva para gestionar tus tareas diarias con un enfoque en el establecimiento de metas y fechas límite.

## 🎯 Objetivo

Crear una herramienta sencilla y efectiva para organizar tareas personales y profesionales, permitiendo al usuario agregar, completar, filtrar y eliminar tareas, con persistencia de datos en el navegador.

## 📋 Requerimientos Cumplidos

- ✅ **Estructura y Maqueta**: HTML, CSS y JS separados. Diseño responsivo y usable.
- ✅ **Interacción con DOM y Eventos**: Manipulación dinámica del DOM con eventos (`click`, `input`, `submit`, `change`).
- ✅ **Formulario y Validaciones**: 4 campos (Título, Descripción, Categoría, Fecha Límite) con 5+ reglas de validación.
- ✅ **Datos y Persistencia**: Almacenamiento en `localStorage` para guardar y recuperar tareas.
- ✅ **Usabilidad y Calidad**: Compatible con múltiples navegadores y tamaños de pantalla. Código limpio y organizado.
- ✅ **Versionamiento**: Repositorio GitHub con commits significativos.

## 🚀 Instrucciones de Ejecución

1. **Clonar o descargar** el repositorio.
2. Abrir el archivo `index.html` en tu navegador favorito.
3. La aplicación cargará automáticamente con datos de ejemplo.
4. ¡Comienza a agregar, completar y organizar tus tareas!

## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS3 (Flexbox, Grid, Animaciones)
- JavaScript (ES6+)
- localStorage API

## 🎨 Funcionalidades

- **Agregar Tarea**: Formulario con validaciones en tiempo real.
- **Lista de Tareas**: Visualización clara con categorías y fechas.
- **Completar/Reabrir**: Cambiar estado de las tareas.
- **Eliminar**: Remover tareas individuales.
- **Filtrar**: Ver tareas por categoría.
- **Persistencia**: Los datos se guardan automáticamente.
